### Introduction
Like Warhammer 40k 10th Edition, games of Warhammer 0th Edition are won by scoring more Victory points than your opponent through various objectives. 

The game consists of 5 Turns called battle rounds, broken into 3 distinct phases:

1. Command Phase

2. Activation Phase

3. Cleanup Phase



### Key Concepts

Unless otherwise stated, all key concepts/terminology is identical to Warhammer 40k 10th Edition

#### Determining Visibility
If any part of another model can be seen from any part of the observing model, it is visible to the observing model so long as neither part overhangs that models base. 

### Unit Characteristics
| M | BS | WS | T | W | Sv |  I | AC | Ld | CL | Wil | Int |

**Movement(M):** How far the model can move. \
**Ballistic Skill(BS):** How well the model can shoot. \
**Weapon Skill(WS):** How well the model can fight in melee. \
**Toughness(T):** How tough the model is(excluding armor), the higher the toughness the less likely they are to be wounded by an attack. \
**Wounds(W):** How much damage the model is able to take before going down.\
**Save(Sv):** The strength of the models armor, the higher the save the less likely they are to be damaged by an attack. \
**Initiative(I):** How quickly the model is able to fight/react in melee \
**Actitivation Count(AC):** The number of activations the unit is allowed to make in the activation phase. \
**Leadership(Ld):** The units ability to issue or follow commands in the heat of the moment. \
**Cool(Cl):** How well the model is able to keep calm under fire. \
**Willpower(Wil):** A measure of the units mental fortitude and willpower in the face of the horrors of 40k. \
**Intelligence(Int):** This represents a fighter's mental acuity and ability to apply knowledge. 



### Command Phase
#### Priority
At the start of the command phase, each player gains 2 CP that they can spend throughout the turn on strategems. Each player then rolls off, with the winner getting priority. In the case of the tie reroll until there is a winner. 
#### Command 
Starting with the player with priority, resolve all command phase rules. Once all command phase rules have been resolved, you can move on to the [Activation Phase](/0thEdition/core/activation/)
